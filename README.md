# Age calculator app solution version 1

Para realizar la primera version de esta calculadora de edad se usa: html, css y javascript.

En HTML se usa 2 inputs, 1 boton, 1 div para mostar el resultado y se llama al script.
En CSS se usa estilos estandar, lo unico novedoso seria un hover sobre el boton Calculate.
En JS se valda que el currentDate no sea nulo, se calcula la edad no haciendo una diferencia del año ingresado, y se hace una validacion adicional para determinar si el usuario aun no cumple años, con lo cual se reduce la edad en 1 año.

Se muestra el resultado en el div con id "currentAge"
